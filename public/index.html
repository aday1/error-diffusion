System Notes

JACK Routing Setup

  • Install 64-bit JACK and register JackRouter.dll as administrator.
  • Run Jack PortAudio and JackControl before opening the DAW environment.
  • Select JackRouter as both input and output device, then route through patchbay.
  • The original notes are Windows-first because that was the actual live transport environment.
Read full setup notes
OSC Controls

Shader And FX Mapping

  • /3/fader1 to /3/fader4 drive the FX functions.
  • /3/fader5 to /3/fader7 handle RGB colour changes.
  • /3/fader8 to /3/fader12 control shifts and global scale.
  • Toggle banks and multitoggle indices select shader states and routing priority.
